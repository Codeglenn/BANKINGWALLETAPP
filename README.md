This is a simple CLI banking application built using Java.
It contains some basic fintech functions like account creation, deposits, withdrawals, balance checking and transaction tracking.

Key features include:
1. Create multiple user accounts
2. Deposit money
3. Withdraw money
4. View account balance
5. Track transaction history
6. Input validation to prevent errors

Future Improvements include:
- Adding database support preferebly POSTGRESQL
- Convert to REST API using Spring Boot
- Add authentication and security
- Deploy as a web service
- Add simpe GUI
